Light trainer - Help page

Mode - 1

Le mode - 1 allume une ou pusieurs Light a la fois une après une sans délais.

Mode - 2

Le mode - 2 allume une ou plusieurs Light avec un temps aléatoire entre le temps min et le temps max insérer en seconde.

Mode - 3

Le mode - 3 allume une light après le temps indiqué même si la précédente n'est pas éteinte.
